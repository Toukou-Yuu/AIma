## 10. 不在 v1 范围内

- **三麻**对局与拔北、北宝牌等（配置里可预埋三麻起点分供未来使用）。
- **包牌、大明杠包**（后续版本）。
- **西入**的**实际局流**（v1 仅开关 + 接口，默认关闭）。
- **地方役、限时活动规则、装饰性 UI**。
- **对局外的 LLM 接口、网络对弈、聊天**：不属于本文牌桌规则。
- **从 Markdown 自动解析规则**：不允许作为运行时依据。

## 11. 配置与默认值（实现须遵守）

下列为产品层约定；具体键名与合并方式由实现定义，但**语义与默认值**应对齐本文。

| 语义 | 默认值 | 说明 |
|------|--------|------|
| 是否启用西入局流 | **false** | v1 不跑西入逻辑；须预留配置与扩展点供后续版本。 |
| 四麻（及当前友人四麻）起配点 | **25000** | 可配置覆盖。 |
| 三麻起配点（未来对局） | **35000** | v1 无三麻对局时可仅占位；启用三麻时默认用此值，可配置。 |
| 一炮多响（多家同时荣和） | **true** | `false` 时的替代规则由实现文档化。 |
| 赤牌方案 | **三赤** | 张数/绑定可配置；默认等同 §4。 |
| 食断（副露后断幺九可役） | **有** | 与雀魂友人常见一致；可与段位场对齐的**配置关闭**由实现提供。 |
